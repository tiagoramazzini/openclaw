Crie um Command Center para OpenClaw em Python + FastAPI (backend) + HTML/CSS/JS vanilla (frontend). É uma UI administrativa que faz proxy para a API do OpenClaw rodando localmente na porta 18789. Arquivo único por página. Dark theme.
Identidade visual:

Background: #0e1117
Cards: #1a1d27
Acento primário: #00b4d8 (azul)
Positivo: #00c853 (verde)
Alerta: #f77f00 (laranja)
Perigo: #d62828 (vermelho)
Fonte: Inter (Google Fonts)

OPENCLAW_TOKEN=your-gateway-token
CC_PORT=8090

Schema SQLite — tabela token_usage:
id, timestamp, agent_id, model, input_tokens, output_tokens,
cost_usd, channel, session_id
Tabela activity_log:
id, timestamp, agent_id, channel, event_type, summary, raw_json
main.py — endpoints FastAPI:
Proxy transparente para OpenClaw:

GET /api/openclaw/status → proxy GET {OPENCLAW_URL}/
GET /api/openclaw/sessions → proxy GET {OPENCLAW_URL}/sessions
POST /api/openclaw/message → proxy POST {OPENCLAW_URL}/v1/chat/completions
GET /api/openclaw/skills → proxy GET {OPENCLAW_URL}/skills
POST /api/openclaw/tools → proxy POST {OPENCLAW_URL}/tools/invoke

Endpoints próprios do Command Center:

GET /api/dashboard → status geral: agentes ativos, mensagens hoje, custo hoje, últimas 10 atividades
GET /api/costs → breakdown de custo por agente/modelo/período (do SQLite)
GET /api/activity → histórico de atividades paginado
POST /api/activity → salvar evento de atividade
GET /api/config → ler ~/.openclaw/openclaw.json e retornar como objeto (mascarar tokens)
PATCH /api/config → escrever seções específicas do config sem sobrescrever o arquivo todo
GET /api/costs/estimate → dado model + tokens estimados, retornar custo em USD e BRL

Tabela de preços por modelo (configurável, valores padrão em R$):
pythonMODEL_PRICES = {

requirements.txt: fastapi, uvicorn, sqlalchemy, httpx, aiofiles, python-multipart
docker-compose.yml:
yamlservices:
  command-center:
    build: .
    ports:
      - "8090:8090"
    volumes:
      - ~/.openclaw:/root/.openclaw:ro  # leitura do config
    environment:
      - OPENCLAW_URL=http://host.docker.internal:18789
      - OPENCLAW_TOKEN=${OPENCLAW_TOKEN}
      - CC_PASSWORD=${CC_PASSWORD:-admin123}
    extra_hosts:
      - "host.docker.internal:host-gateway"
Notas importantes:

O Command Center nunca expõe o OPENCLAW_TOKEN para o frontend — toda chamada passa pelo backend FastAPI
Leitura do openclaw.json é somente leitura por padrão — escrita requer confirmação explícita
Se o Gateway estiver offline, mostrar banner vermelho fixo no topo de todas as páginas
Polling do activity feed a cada 5s — parar polling se aba estiver em background (Page Visibility API)
Todos os custos em R$ com 4 casas decimais (ex: R$ 0,0023) para não perder precisão em valores pequenos
